Detailed Description

Parameter storage.

Stores parameter value.

APM uses:

  • int8 (primary for bools)
  • int16 ???
  • int32 for int's
  • real32 for float's

PX4:

  • int32 for int's
  • real32 for float's

So no reason to really use boost::any. But feel free to fire an issue if your AP do not like it.
