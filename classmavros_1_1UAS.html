Detailed Description

UAS for plugins.

This class stores some useful data and provides fcu connection, mode stringify utilities.

Currently it stores:

  • FCU link interface
  • FCU System & Component ID pair
  • Connection status (mavplugin::SystemStatusPlugin)
  • Autopilot type (mavplugin::SystemStatusPlugin)
  • Vehicle type (mavplugin::SystemStatusPlugin)
  • IMU data (mavplugin::IMUPubPlugin)
  • GPS data (mavplugin::GPSPlugin)
